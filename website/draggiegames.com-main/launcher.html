The Draggie Games Launcher

The Draggie Games Launcher, or client, is a lightweight, optimised application that allows you to download, play and manage a whole host of Draggie Games, including Project Saturnian and so many more on your Windows device.

Constantly updated with new features, games and more, the Draggie Games Launcher is the best way to play Draggie Games.

Information

About the Launcher

The Draggie Games Launcher is currently a command-line application that allows you to download, play and manage your access to all of our games. It is currently in beta, and is available for Windows only.

Please note! We issue updates to the Draggie Games launcher regularly. If you are having issues with the client, please allow the program to update to the latest version. You can check the version number in the top of the Launcher's window.

Whilst the application is built inside a command-line interface, it should still be easy to use. On top of this, in version 1.0.0, we will be revamping the Launcher with a brand new, graphical UI, making it even easier to use!

Addons

Addons and Extras

Draggie Games Addons are a way to enhance your experience. Addons can be downloaded from the Addons page on this website, or directly from the Launcher. These can be used to further optimise your experience, by allowing you to auto-update your games, change how the Launcher looks and responds, and manage your account's items, including locker items for Project Saturnian.

Addons are currently in testing; features may be added or removed at any time. If you experience any issues, contact us, and if you have any suggestions for more addons, please let us know!
