What is Project Saturnian?

Project Saturnian is a game showcases the power of Draggie Games' developer APIs, includes crazy water physics features, and is... fun!

As our first game with these features built-in, it is in early Alpha testing. You can redeem your code now to get early access!

Project Saturnian seamlessly integrates with the Draggie Games launcher and the Draggie Games accounts, locker and auto-update systems.
